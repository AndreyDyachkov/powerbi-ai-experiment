# AI Generated Power BI Churn Dashboard Project 
## Project Overview
This project is a Power BI dashboard designed to analyze and visualize customer churn data using AI assistance. The objective is to test the ability of AI to build relevant dashboards provide insights into the factors contributing to customer churn.

## Project Structure

- **/PBIP**: Contains Power BI project.
- **/Visuals_descriptions**: Contains descriptions of visuals used in the reports.
- **Churn_dashboard.pdf**: The main dashboard report exported in PDF format.
